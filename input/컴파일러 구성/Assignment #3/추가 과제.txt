Assignment #3 U-Code 프로그램 구현 및 인터프리터 실습

추가과제

U-Code로 마방진(Magic Number)를 출력하는 프로그램을 작성하고 실험하시오.

  마방진을 계산하는 프로그램은 3~15 사이의 홀수를 사용자로 부터 입력 받고, 마방진을 결과로 출력함.

  0 을 입력받으면 프로그램이 종료됨.

  홀수 입력시 해당하는 마방진을 출력하고 다시 숫자를 입력받을 수 있도록 함.

  짝수를 입력 받으면 무시하고 다시 숫자를 입력 받을 수 있도록 함.

추가과제를 구현했을 경우, 레포트 표지에 반드시 명시.
